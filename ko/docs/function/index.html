매처의 함수​

매처에 함수를 지정하여 더 강력하게 만들 수 있습니다. 내장 함수를 사용하거나 자신의 함수를 지정할 수 있습니다. 내장 키 매칭 함수는 다음 형식을 사용합니다:

bool function_name(string url, string pattern)

url이 pattern과 일치하는지 여부를 나타내는 부울을 반환합니다.

지원되는 내장 함수는 다음과 같습니다:

함수url패턴예시
keyMatch/alice_data/resource1과 같은 URL 경로/alice_data/*와 같은 URL 경로 또는 * 패턴keymatch_model.conf/keymatch_policy.csv
keyMatch2/alice_data/resource1과 같은 URL 경로/alice_data/:resource와 같은 URL 경로 또는 : 패턴keymatch2_model.conf/keymatch2_policy.csv
keyMatch3/alice_data/resource1과 같은 URL 경로/alice_data/{resource}와 같은 URL 경로 또는 {} 패턴https://github.com/casbin/casbin/blob/277c1a2b85698272f764d71a94d2595a8d425915/util/builtin_operators_test.go#L171-L196
keyMatch4/alice_data/123/book/123과 같은 URL 경로/alice_data/{id}/book/{id}와 같은 URL 경로 또는 {} 패턴https://github.com/casbin/casbin/blob/277c1a2b85698272f764d71a94d2595a8d425915/util/builtin_operators_test.go#L208-L222
keyMatch5/alice_data/123/?status=1과 같은 URL 경로/alice_data/{id}/*와 같은 URL 경로, {} 또는 * 패턴https://github.com/casbin/casbin/blob/1cde2646d10ad1190c0d784c3a1c0e1ace1b5bc9/util/builtin_operators_test.go#L485-L526
regexMatch아무 문자열정규 표현식 패턴keymatch_model.conf/keymatch_policy.csv
ipMatch192.168.2.123과 같은 IP 주소192.168.2.0/24와 같은 IP 주소 또는 CIDRipmatch_model.conf/ipmatch_policy.csv
globMatch/alice_data/resource1과 같은 경로 형식의 경로/alice_data/*와 같은 glob 패턴https://github.com/casbin/casbin/blob/277c1a2b85698272f764d71a94d2595a8d425915/util/builtin_operators_test.go#L426-L466

키를 가져오는 함수들은 보통 세 개의 매개변수를 받습니다(keyGet 제외):

bool function_name(string url, string pattern, string key_name)

패턴과 일치하는 경우 키 key_name의 값을 반환하고, 일치하는 것이 없으면 ""을 반환합니다.

예를 들어, KeyGet2("/resource1/action", "/:res/action", "res")는 "resource1"을 반환하고, KeyGet3("/resource1_admin/action", "/{res}_admin/*", "res")는 "resource1"을 반환합니다. KeyGet에 대해서는, 두 개의 매개변수를 받으며, KeyGet("/resource1/action", "/*)는 "resource1/action"을 반환합니다.

함수url패턴key_name예시
keyGet/proj/resource1과 같은 URL 경로/proj/*와 같은 URL 경로 또는 * 패턴\keyget_model.conf/keymatch_policy.csv
keyGet2/proj/resource1과 같은 URL 경로/prooj/:resource와 같은 URL 경로 또는 : 패턴패턴에 지정된 키 이름keyget2_model.conf/keymatch2_policy.csv
keyGet3/proj/res3_admin/과 같은 URL 경로/proj/{resource}_admin/*와 같은 URL 경로 또는 {} 패턴패턴에 지정된 키 이름https://github.com/casbin/casbin/blob/7bd496f94f5a2739a392d333a9aaaa10ae397673/util/builtin_operators_test.go#L209-L247

사용자 정의 함수 추가 방법​

먼저, 함수를 준비합니다. 여러 매개변수를 받아 bool을 반환합니다:

func KeyMatch(key1 string, key2 string) bool {
    i := strings.Index(key2, "*")
    if i == -1 {
        return key1 == key2
    }

    if len(key1) > i {
        return key1[:i] == key2[:i]
    }
    return key1 == key2[:i]
}

그런 다음, interface{} 타입으로 감싸십시오:

func KeyMatchFunc(args ...interface{}) (interface{}, error) {
    name1 := args[0].(string)
    name2 := args[1].(string)

    return (bool)(KeyMatch(name1, name2)), nil
}

마지막으로, 함수를 Casbin enforcer에 등록합니다:

e.AddFunction("my_func", KeyMatchFunc)

이제, 모델 CONF에서 이 함수를 다음과 같이 사용할 수 있습니다:

[matchers]
m = r.sub == p.sub && my_func(r.obj, p.obj) && r.act == p.act
