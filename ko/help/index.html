도움이 필요한가요?

Casbin을 사용하는 중에 도움이 필요하다면, 아래에 있는 방법 중 하나를 시도해보세요.



포럼

Casbin 포럼에서 관리자와 토론하거나 Casbin에 대한 경험을 공유하세요.

Discord

디스코드에서 대화에 참여하여 도움을 기여할 수 있습니다.

텐센트 QQ

QQ 그룹에 가입하여 저희에게 연락하거나 546057381 또는 WeChat group을 이용할 수 있습니다.

Github

GitHub 저장소에서 찾아보고 이슈 또는 풀 요청를 제출하여 발견한 버그나 구현하고 싶은 새로운 기능을 알려주세요.
